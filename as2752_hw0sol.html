Problem 2.1

If $a > \sqrt{x}$, then $\frac{x}{a} < \frac{x}{\sqrt{x}}$ or $\frac{x}{a} < \sqrt{x}$. If $a < \sqrt{x}$, then $\frac{x}{a} > \frac{x}{\sqrt{x}}$ or $\frac{x}{a} > \sqrt{x}$. In either case, $\sqrt{x}$ is larger than one of $\frac{x}{a}$ and $a$ and smaller than the other.

julia> function p_dynamics(a, yt, b, q, T, X0)
         X = zeros(T);
         X[1] = X0
         for i in 1:T-1
           X[i+1] = X[i] + a + yt[i] + X[i]^q/(1+X[i]^q) - b*X[i]
         end
         return X
       end
